How To Read The Cost Curve
The cost curve is the marginal cost of production — what it costs to mine the last unit. When spot price is 1–2x the cost curve, we're at cycle lows (buy zone). When it reaches 4–5x, we're approaching historic peaks (reduce). This is the framework — not financial advice.
1–2x = Cycle Low / Buy Zone
2–4x = Mid Cycle / Hold
4–5x+ = Cycle Peak / Reduce
